## 🚀 精选项目

* **[人生记录轴 (Record Timeline)](https://github.com/duanxinyua/record-timeline)**
  * 📖 **简介**: 一款帮助你记录人生重要时刻的应用。它不只是一个日历或日记，更是你专属的时间胶囊。
  * ✨ **亮点**: 
    * 优雅的时间轴展示，让回忆一目了然。
    * 支持图文并茂的多媒体分组发布记录。
    * 自动提取照片 EXIF 信息（拍摄时间、GPS位置）。
  * 🛠️ **技术栈**: `uni-app`, `Vue 3`, `Vite`, `PHP`, `SQLite`

* **[女巫的毒药 (WitchPoison)](https://github.com/duanxinyua/WitchPoison)**
  * 📖 **简介**: 创意十足的在线多人回合制心理博弈小游戏。融合了趣味与技术。
  * ✨ **亮点**:
    * 独特的博弈算法设计，支持2-5人在线对战。
    * 实时互动，直观的游戏可视化体验。
  * 🛠️ **技术栈**: `UniApp`, `Vue 3`, `Node.js`, `Express`, `WebSocket`

## 🛠️ 技术栈概览

- **前端:** Vue 3, uni-app, Vite, HTML/CSS/JavaScript
- **后端:** Node.js, Express, PHP
- **数据库:** SQLite
- **运维与部署:** Linux 环境高可用部署与性能调优
- **其他技术:** WebSocket (实时通信), 媒体处理 (EXIF提取, GD库图像处理)
